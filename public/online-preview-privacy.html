Online preview privacy & safety

A small, invitation-only tank battle. No accounts, no arbitrary chat, and no application session telemetry in the public preview.

What the game handles

The server keeps your callsign, private room state, team, inputs, and reconnection reservation in memory while the room is active. That room data disappears when the room is cleaned up or the server restarts.

Your browser keeps a callsign and short-lived reconnection information for the current browser session. You can remove it by closing the browser session or clearing this site's data.

What is not recorded

Application-level session telemetry is disabled for the public preview. The game does not create a production log of room keys, callsigns, IP addresses, positions, inputs, snapshots, radio activity, or results.

The hosting and network providers still handle connection metadata to deliver the service and may retain infrastructure security or access logs under their own controls. The preview invitation must identify the selected host and current provider notice before play.

Communication

There is no arbitrary player chat. In-game communication is limited to five built-in radio commands and team map pings. Longer conversation belongs in the third-party channel used to organize the match.

Callsigns and abuse

Callsigns are anonymous display labels, not verified identities. The game removes markup, emoji, controls, invisible formatting, and overlong text, but it does not attempt to guess every offensive word.

  • In the lobby, ask the host to kick an abusive callsign.
  • During a match, leave if continuing is uncomfortable.
  • Report the callsign, approximate UTC time, and category through the same private channel that carried your invitation.
  • Do not send a room key, reconnection token, IP address, or telemetry file.

Questions or requests

The invitation must name the maintainer contact for this preview. Use that private contact for an abuse report, access/deletion question, or suspected data incident. There is no public report form or automated account-data export because the preview has no accounts or permanent player profiles.
